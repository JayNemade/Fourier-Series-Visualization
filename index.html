Fourier Series

Introduction

This project is on the demonstration of fourier series by visulizing the finite-infinite sum of sines and cosines. The concept behind the fourier series was given by a brilliant mathematician by the name of, Joseph Fourier. The idea of the original paper was that any arbitrary periodic function could be represented by a specific sum of sines and cosines.

The theme behind this webpage is that even a layman should understand the key concepts behind the fourier series so the language would be simplistic and using 'complex' mathematical concepts will be avoided. If you want to know more about the topic citations will be provided. If you are already familiar with fourier series, you can skip to Instructions for use.

Basics of trignometry

Before understanding the fourier series we must know what is a sin() and a cos() function.

Sine wave/function

Mathematically the sine of an angle in a right angled triangle is the ratio of it's opposite side and it's hypotenuse. Thus sine is a function of a variable angle. In this project we look at the graph of this function as the position of a point on a moving circle. This is shown below or, you can view an animated version by sliding the slider for n to 1.


Cosine wave/function

Like wise the cosine function is obtained from the ratio of the Adjacent side and the hypotenuse. It is the sine function itself but with a phase shifted of 90 degrees.


Relationship between the osscilation and the sine wave produced by it

The Sine wave has some properties which we should be aware of. The sine wave has a wavelength λ and an Amplitude A.
Wavelength λ is defined as the time required for the function to complete one full osscilation. Thus λ is inversly proportional to the speed of the osscilation or the time period.
The Amplitude A is defined as the height of the wave. Therefore A is directly proportional to the radius.

Concept behind the Fourier series

A Fourier series [1] is an expansion of a periodic function [2] f (x) in terms of an infinite sum of sines and cosines. Fourier series make use of the orthogonality [3] relationships of the sine and cosine functions, i.e., their graphs at any point intersect at right angle. The computation and study of Fourier series is known as harmonic analysis.

Where is the use of this project

The sole use of this project is in academics. Professors can use this visualization if they are teaching harmonic analysis.
